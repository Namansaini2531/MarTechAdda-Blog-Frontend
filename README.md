# MarTechAdda Blog Frontend

Welcome to the frontend repository for the **MarTechAdda Blog**. This project powers the high-performance, dynamic user interface for MarTechAdda's editorial and marketing content platform.

## 🚀 Tech Stack

- **Framework**: [React](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Routing**: [React Router DOM](https://reactrouter.com/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)

## ✨ Key Features

- **Responsive Design**: Flawless experience across desktop, tablet, and mobile devices.
- **Dynamic Routing**: Seamless navigation between the home page, blog listings, and individual blog posts.
- **Modern UI/UX**: Custom Tailwind configuration for a premium, readable, and engaging aesthetic, drawing inspiration from top-tier SaaS and editorial websites.
- **Performance Optimized**: Fast loads with Vite's optimized build pipeline.

## 🛠️ Getting Started

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Namansaini2531/MarTechAdda-Blog-Frontend.git
   cd MarTechAdda-Blog-Frontend
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

### Development Server

Run the local Vite development server to view and test changes:

```bash
npm run dev
```

Open `http://localhost:5173` in your browser.

### Building for Production

To create a production-ready build:

```bash
npm run build
```

The compiled assets will be placed in the `dist` directory.

## 🚢 Deployment

This project is configured to be easily deployed on platforms like **Vercel** or **Netlify**. Simply connect your GitHub repository and set the framework preset to `Vite`.
